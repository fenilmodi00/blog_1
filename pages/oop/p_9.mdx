#### 3 Write program to create two threads .One using Thread class and another using Runnable Interface.

```java filename="P9_3.java" copy
public class P9_3 {
    public static void main(String[] args) {
        th1 t1 = new th1();
        th2 t2 = new th2();
        Thread t = new Thread(t2);
        t1.start();
        t.start();
    }
}
class th1 extends Thread{
    public void run()
    {
        for(int i=1;i<=5;i++)
        {
            System.out.println("Thread 1"+i);
        }
        try
        {
            Thread.sleep(500);
        }
        catch (InterruptedException ex)
        {
            System.out.println("Thread Interrupted");
         }
    }
}
class th2 implements Runnable
{
    public void run()
    {
        for(int i=1;i<=5;i++)
        {
            System.out.println("Thread 2"+i);
        }
        try
        {
                Thread.sleep(5000);
         }
        catch (InterruptedException ex)
        {
            System.out.println("Thread Interrupted");
         }
    }
}
```

#### 4 ‚¶Å Write program to create two threads .One thread prints table of 300 and another thread prints table of 500.

```java filename="P9_4.java" copy
public class P9_4
{
    public static void main(String[] args)
    {
        main1 a = new main1();
        main2 b = new main2();
        Thread t = new Thread(b);
        a.run();
        b.run();
    }
}
class main1 extends Thread
{
    public void run()
    {
        int i;

        for(i=1;i<=10;i++)
        {
            System.out.println(" 300 x "+i+" = "+300*i);
         }
    }
}
class main2 extends Thread
{
    public void run()
    {
        int i;
        for(i=1;i<=10;i++)
        {
                System.out.println(" 500 x "+i+" = "+500*i);
        }
    }
}
```
